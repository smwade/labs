# Labs

Experiments, demos, and web apps — [labs.seanwade.com](https://labs.seanwade.com)

## Adding a demo

**Plain HTML:** Drop a folder in `public/demos/`, add a card to `app/page.tsx`, push.

**React page:** Create `app/demos/my-demo/page.tsx`, add a card, push.

**With API route:** Create `app/api/my-demo/route.ts` for the backend, plus the frontend above.
